# 旧暦カレンダー 2026年3月

## 概要

2026年（令和8年・丙午年）3月を旧暦情報付きで表示するシングルページカレンダー。
HTML + CSS + JavaScript（バニラ）のみで構成。

## ファイル構成

```
calender/
└── index.html   # カレンダー本体（HTML/CSS/JS 一体型）
```

## 実装内容

### 旧暦変換

| 新暦 | 旧暦 |
|------|------|
| 3/1 〜 3/17 | 旧暦 正月（1月）13日 〜 29日 |
| 3/18 〜 3/31 | 旧暦 如月（2月）1日 〜 14日 |

- 旧暦1月1日 = 2026/2/17（新月）、1月は29日（小の月）
- 旧暦2月1日 = 2026/3/18（新月）

### 六曜

計算式：`(旧暦月 + 旧暦日) % 6`

| index | 六曜 | 意味 |
|-------|------|------|
| 0 | 大安 | 最も吉 |
| 1 | 赤口 | 凶日（正午のみ吉） |
| 2 | 先勝 | 午前吉・午後凶 |
| 3 | 友引 | 友を引く（葬儀を避ける） |
| 4 | 先負 | 午前凶・午後吉 |
| 5 | 仏滅 | 最も凶 |

### 月の満ち欠け

| 日付 | 旧暦 | 月相 |
|------|------|------|
| 3/3  | 1月15日 | 🌕 満月 |
| 3/10 | 1月22日 | 🌗 下弦 |
| 3/18 | 2月1日  | 🌑 新月 |
| 3/25 | 2月8日  | 🌓 上弦 |

### 二十四節気

| 日付 | 節気 |
|------|------|
| 3/6  | 啓蟄（けいちつ）- 虫が冬眠から目覚める頃 |
| 3/20 | 春分（しゅんぶん）- 昼夜の長さが等しくなる |

### 祝日

| 日付 | 祝日 |
|------|------|
| 3/20（金）| 春分の日 |

## 表示仕様

- 日曜・祝日：日付を赤色表示
- 土曜：日付を青色表示
- 旧暦の月初（1日）：青字・太字で強調
- 六曜：種類ごとに色分けバッジ
- 今日の日付：黄色背景＋「今日」バッジ（2026/3 を閲覧時のみ）
